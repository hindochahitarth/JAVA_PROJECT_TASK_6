A simple Java Swing application for managing your daily tasks.

## Features

- ✅ Add new tasks
- ✅ Delete selected tasks
- ✅ Clean and simple interface


## How to Use

1. **Adding a task:**
   - Type your task in the text field
   - Click "Add" button or press Enter

2. **Deleting a task:**
   - Select a task from the list
   - Click "Delete" button

## Requirements

- Java JDK 8 or later
